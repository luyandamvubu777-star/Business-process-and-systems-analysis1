# As-Is Process: IT Support Request

## Description
This document describes the current informal process used by staff
to request IT support within an organisation.

## Current Steps
1. Staff member reports an issue verbally or via WhatsApp.
2. No formal record of the issue is created.
3. IT support responds when available.
4. Resolution is not documented.
5. Repeated issues are not tracked.

## Identified Problems
- No accountability
- No issue tracking
- Loss of information
- Delays in response
