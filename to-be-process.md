# To-Be Process: IT Support Request

## Description
This document outlines a structured and traceable process for
handling IT support requests.

## Improved Steps
1. Staff member submits a support request via a digital form.
2. Request is automatically logged with date and issue category.
3. IT support receives notification.
4. Issue status is updated during resolution.
5. Resolved issues are stored for future reference.

## Benefits
- Clear accountability
- Traceable history
- Improved response times
- Data for continuous improvement
